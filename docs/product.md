# Product

## Product Summary

ApplyPilot is a personal job-search CRM for active job seekers who want a clearer way to track applications, contacts, interview progress, and follow-ups.

## MVP Problem Statement

Job seekers often manage applications across spreadsheets, notes, emails, and memory. The MVP should provide one place to track the search process without adding complexity.

## MVP Users

- individual job seekers
- people applying to multiple roles at once
- users who want a lightweight personal workflow, not a team ATS

## Core MVP Jobs To Be Done

- Track job applications and their status
- Store contacts related to each opportunity
- Track interview stages
- Record follow-up dates and notes

## MVP Non-Goals

- AI features
- Gmail integration
- calendar integration
- browser extension
- payments
- mobile app
- team accounts
- scraping

## Success For The First Version

The MVP is successful if a user can:

- sign in
- create and update applications
- attach contacts and notes to applications
- manage interview stages
- review pending follow-ups
